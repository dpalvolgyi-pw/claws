# 보안 정책

## 지원 버전

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## 취약점 보고

claws에서 보안 취약점을 발견하신 경우, [GitHub Issue 생성](https://github.com/clawscli/claws/issues/new)을 통해 보고해 주십시오.

보고 시 다음 내용을 포함해 주십시오:
- 취약점에 대한 설명
- 문제를 재현하는 단계
- 관련 로그 또는 스크린샷

가능한 한 빠르게 응답하고, 문제 해결을 위해 협력하겠습니다.
